## Account Creation

<Steps>
  <Step title="Sign Up for a User account">
    Register and create your personal user account with [Cerevox](https://cerevox.ai/app)
  </Step>
  <Step title="Create a Cerevox Account">
    Establish a Cerevox Account to access our platform's full range data normalization features.
  </Step>
  <Step title="Provision Keys">
    Generate a secure API key for your Cerevox Account to start normalizing your data. Remove keys that have been compromised.
  </Step>
</Steps>

### Data Normalization

<AccordionGroup>
  <Accordion icon="quinscape" title="API">
    Normalize documents through data.cerevox.ai

    ```bash API Call Example
    curl -X 'POST' \
    'https://www.data.cerevox.ai/v0/elements' -k \
    -H 'accept: application/json' \
    -H 'Content-Type: multipart/form-data' \
    -H 'cerevox-api-key: <YOUR-KEY>' \
    -F 'files=@sample.pdf'
    ```
