Normalize documents through data.cerevox.ai
    
```bash API Call Example
curl -X 'POST' \
'https://www.data.cerevox.ai/v0/elements' -k \
-H 'accept: application/json' \
-H 'Content-Type: multipart/form-data' \
-H 'cerevox-api-key: <YOUR-KEY>' \
-F 'files=@sample.pdf'
```
